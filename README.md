<h2>Install</h2>
<h3>Windows</h3>

```sh
# clone repo... (use ssh or https)
cd test-game
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

<h3>Linux</h3>

```sh
# clone repo... (use ssh or https)
cd test-game
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
